# Fashion Hub

Fashion Hub is a full-stack online clothing store that enables customers to explore fashion products, view product details, add items to their cart, and place orders through a modern and user-friendly interface.

## Tech Stack

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Spring Boot
* Java
* REST APIs

### Database

* MySQL

## Features

* User Registration and Login
* Browse Men's, Women's, and Kids' Fashion Collections
* View Product Details and Pricing
* Search and Filter Clothing Products
* Shopping Cart Management
* Secure Order Placement
* Responsive Design for Mobile and Desktop Devices

## Project Structure

Fashion-Hub/

├── frontend/   (React.js Application)

└── backend/    (Spring Boot REST API)

## How to Run

### Frontend

cd frontend

npm install

npm start

### Backend

cd backend

mvn spring-boot:run
